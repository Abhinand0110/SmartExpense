SmartExpense

SmartExpense is a personal finance management web application that I built to help users track their income and expenses in a simple and organized way. The main idea behind this project is to give users a clear understanding of where their money is going and help them make better financial decisions.

What this project does

The user first registers or logs into the system. After logging in, they are taken to a welcome/workspace page, which acts as a starting point for the application.

From there, the user enters the main workspace where they can:

How the system works (simple flow)
User logs in or registers
User enters the workspace
Adds income and expenses
Data is stored in the database
Dashboard fetches the data and shows insights
User can compare months and analyze savings
AI assistant reads the data and gives suggestions
Database (basic idea)

The project mainly uses these tables:

Users → stores user details
Income → stores income entries
Expenses → stores expense entries
Compare → stores calculated data for comparisons
Feedback → stores user feedback
Note

For now, passwords are stored in plain text because this is an academic project. In a real-world application, passwords should be encrypted using hashing techniques.
